# 成语接龙 · 雅集

一个纯本地运行的极简风“成语接龙”网页游戏。

## 🎮 游戏规则

1. **倒计时**：每次接龙有 **30 秒** 的思考时间，超时即游戏结束。
2. **输入**：在输入框内输入四字成语（可带或不带标点），按下回车或点击“落笔”。
3. **校验**：输入的成语必须存在于本地词库中，且首字/首音必须与前一个成语的尾字/尾音匹配。
4. **得分机制**：
   - 接得越快得分越高。
   - 匹配度越高得分倍率越高（**字同** 1.5倍 > **调同** 1.2倍 > **音同** 1.0倍），支持叠加连击！

## 🚀 如何开始

本项目为纯静态网页，**无需任何环境配置**，双击即玩：

1. 克隆或下载本项目到本地。
2. 找到根目录下的 `index.html` 文件。
3. 双击使用任意现代浏览器打开即可游玩。
